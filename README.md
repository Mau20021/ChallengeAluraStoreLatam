# 📊 Challenge Alura Store LATAM

## 📌 Descripción del Proyecto

Análisis de datos realizado sobre las cuatro tiendas de la cadena **Alura Store** con el objetivo de determinar cuál presenta el menor desempeño para recomendar su venta estratégica.

Se evaluaron métricas clave como ingresos totales, ventas por categoría, calificaciones de clientes, productos más vendidos y costo promedio de envío.

---

## 🎯 Objetivo

Identificar la tienda menos eficiente mediante un análisis comparativo basado en datos y presentar una recomendación sustentada cuantitativamente.

---

## 🛠 Herramientas Utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

---

## 📊 Principales Hallazgos

- La **Tienda 4** presenta la menor facturación total.
- Tiene menores ingresos en las categorías más importantes.
- No destaca en calificaciones de clientes.
- Su menor costo de envío no compensa el bajo volumen de ventas.

---

## 🧠 Conclusión

Con base en el análisis realizado, se recomienda vender la **Tienda 4**, ya que es la unidad con menor desempeño global y su desinversión tendría el menor impacto negativo en la cadena.
